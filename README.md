# School-Management-System
A complete School Management System using Python

- This project is a command-line based School Enterprise Resource Planning (ERP) system designed to digitize and streamline the daily administrative and academic tasks of "UHD-Schools." The system provides a centralized platform for managing students, teachers, attendance, and academic results.
- The application features three distinct portals—Admin, Teacher, and Student—each with role-specific functionalities. It utilizes JSON for persistent data storage, generates visual performance charts using Matplotlib, and produces official student result cards as PDF documents.
